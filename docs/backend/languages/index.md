# Backend Languages

Esta seccion agrupa lenguajes y plataformas usadas frecuentemente en backend para estudiar fortalezas, tradeoffs y patrones comunes.

## Lenguajes y plataformas

- [Dotnet](./dotnet/) - Desarrollo backend sobre CLR y ecosistema Microsoft.
- [Go](./go/) - Servicios concurrentes y binarios simples de operar.
- [Java](./java/) - Backend empresarial y ecosistema maduro.
- [Node.js](./nodejs/) - Servicios JavaScript y TypeScript orientados a I/O.
- [Python](./python/) - APIs, automatizacion y servicios de rapida iteracion.
