# Backend Frameworks

Esta seccion organiza frameworks backend por ecosistema para comparar ergonomia, arquitectura y casos de uso.

## Frameworks

- [ASP.NET Core](./aspnet-core/) - Desarrollo backend en el ecosistema .NET.
- [Django](./django/) - Framework Python opinionado para aplicaciones completas.
- [Express](./express/) - Base minimalista para APIs en Node.js.
- [FastAPI](./fast-api/) - APIs tipadas y asincronas en Python.
- [Fastify](./fastify/) - Framework Node.js orientado a performance.
- [Flask](./flask/) - Microframework Python para servicios ligeros.
- [NestJS](./nestjs/) - Arquitectura modular para aplicaciones Node.js.
- [Spring Boot](./spring-boot/) - Ecosistema Java para aplicaciones empresariales.
