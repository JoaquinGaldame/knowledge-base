# Software Engineering Fundamentals

## Introducción

Esta sección reúne conceptos teóricos y académicos sobre Ingeniería de Software desde la perspectiva de las Ciencias de la Computación.

El objetivo es documentar fundamentos, definiciones, modelos, principios y conocimientos base que permiten comprender la Ingeniería de Software como disciplina, más allá de tecnologías, frameworks o herramientas específicas.
