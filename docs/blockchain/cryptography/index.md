# Blockchain Cryptography

Esta seccion cubre las primitivas criptograficas que sostienen identidad, integridad y verificabilidad en blockchain.

## Documentos principales

- [Hashing](./hashing.md) - Integridad de datos y huellas criptograficas.
- [Public Private Keys](./public-private-keys.md) - Propiedad y control de cuentas.
- [Digital Signatures](./digital-signatures.md) - Autenticidad y autorizacion de operaciones.
- [Elliptic Curves](./elliptic-curves.md) - Base matematica de firmas modernas.
- [Merkle Trees](./merkle-trees.md) - Verificacion eficiente de conjuntos de datos.
- [ZK Proofs](./zk-proofs.md) - Pruebas criptograficas con privacidad.
