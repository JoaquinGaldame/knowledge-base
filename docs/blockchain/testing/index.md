# Blockchain Testing

Esta seccion cubre estrategias para validar contratos y sistemas Web3 en multiples niveles.

## Documentos principales

- [Unit Testing](./unit-testing.md) - Validacion aislada de funciones y reglas.
- [Integration Testing](./integration-testing.md) - Flujos entre contratos y componentes.
- [Security Testing](./security-testing.md) - Pruebas orientadas a vulnerabilidades.
- [Fuzz Testing](./fuzz-testing.md) - Exploracion automatica de entradas inesperadas.
- [Invariant Testing](./invariant-testing.md) - Propiedades que siempre deben cumplirse.
- [Fork Testing](./fork-testing.md) - Escenarios realistas sobre estado de red.
- [Foundry Testing](./foundry-testing.md) - Herramientas y enfoque de pruebas con Foundry.
