# Roadmaps

Esta seccion organiza rutas de aprendizaje para recorrer la biblioteca con una progresion clara y acumulativa.

## Areas de la biblioteca

- [Testing](../testing/roadmap.md) - Recorrido de aprendizaje para construir criterio profesional en testing.
- [Architecture](../architecture/) - Estilos arquitectonicos, decisiones tecnicas y tradeoffs.
- [System Design](../system-design/) - Escalabilidad, consistencia, mensajeria y observabilidad.
- [Backend](../backend/) - APIs, patrones, frameworks, lenguajes y buenas practicas.
- [Frontend](../frontend/) - Interfaces, frameworks y calidad de experiencia.
- [Databases](../databases/) - Motores, rendimiento, transacciones e indexacion.
- [Cloud](../cloud/) - Infraestructura, despliegues, contenedores y operacion.
- [Blockchain](../blockchain/) - Fundamentos, contratos, seguridad y proyectos Web3.
- [Computer Science](../computer-science/) - Bases teoricas para fortalecer criterio ingenieril.
- [Software Engineering](../software-engineer/) - Patrones y conceptos transversales de construccion de software.
- [Projects](../projects/) - Casos practicos, incidentes y lecciones aprendidas.
