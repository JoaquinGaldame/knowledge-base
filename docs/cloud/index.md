# Cloud & DevOps

Esta seccion agrupa infraestructura, despliegues, contenedores, observabilidad y automatizacion operativa.

## Secciones principales

- [AWS](./aws/) - Servicios base para aplicaciones en la nube.
- [Docker](./docker/) - Contenedorizacion y empaquetado reproducible.
- [Kubernetes](./kubernetes/) - Orquestacion y operacion de workloads.
- [Observability](./observability/) - Logs, metricas, trazas y diagnostico.
- [Terraform](./terraform/) - Infraestructura como codigo y aprovisionamiento.
