# AWS

Esta seccion agrupa servicios base de AWS relevantes para despliegues de aplicaciones y operacion de infraestructura.

## Servicios

- [ALB](./alb/) - Balanceo HTTP y enrutamiento de trafico.
- [CloudWatch](./cloud-watch/) - Logs, metricas y alarmas.
- [EC2](./ec2/) - Computo general en maquinas virtuales.
- [ECS](./ecs/) - Contenedores administrados en AWS.
- [IAM Roles](./iam-roles/) - Permisos y acceso entre recursos.
- [RDS](./rds/) - Bases de datos relacionales administradas.
- [Secrets Manager](./secrets-manager/) - Gestion segura de secretos y credenciales.
