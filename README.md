# Engineering Knowledge Base

Personal engineering knowledge base focused on software testing, software architecture, backend development, frontend development, cloud infrastructure and distributed systems.

This repository contains a curated collection of technical documentation, study notes, practical guides and engineering references built to consolidate long-term knowledge and support continuous professional growth.

The project is published as a documentation website using VitePress.

---

## Purpose

The primary goal of this project is to create a structured and maintainable engineering library that transforms scattered learning into reusable knowledge.

The content is intended to:

* Document fundamental software engineering concepts.
* Deepen understanding of software testing and quality assurance.
* Capture architecture and design knowledge.
* Preserve practical lessons learned from real-world projects.
* Serve as a personal reference system.
* Share valuable engineering knowledge with other developers.

## Local Development

Install dependencies:

```bash
npm install
```

Start the documentation server:

```bash
npm run docs:dev
```

The site will be available at:

```txt
http://localhost:5173
```

---

## Build

Generate the static website:

```bash
npm run docs:build
```

Preview the generated build:

```bash
npm run docs:preview
```

## Learning Philosophy

This repository is not intended to be a collection of quick notes.

The objective is to build deep understanding through:

* Verified sources.
* Practical examples.
* Real-world applications.
* Critical thinking.
* Continuous refinement.

Knowledge should be treated as a long-term engineering asset rather than temporary study material.
